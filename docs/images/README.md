# Documentation Images

This directory contains images and screenshots for the AI Chatbot project documentation.

## Files

### `ai-chatbot-screenshot.png`
- **Purpose**: Main application screenshot for README.md
- **Description**: Full-screen view of the AI Chatbot interface showing all three main sections (sidebar, chat area, file upload)
- **Usage**: Referenced in the main README.md file to showcase the application interface

## Image Guidelines

### Screenshot Requirements
- **Resolution**: Minimum 1920x1080 for desktop screenshots
- **Format**: PNG preferred for screenshots with transparency
- **Quality**: High quality, clear text and interface elements
- **Content**: Show the complete application interface with all sections visible

### File Naming
- Use descriptive names with hyphens: `ai-chatbot-screenshot.png`
- Include version numbers if needed: `interface-v2-screenshot.png`
- Keep names lowercase and descriptive

### Usage in Documentation
- Reference images with relative paths: `docs/images/filename.png`
- Include alt text for accessibility
- Provide context about what the image shows

## Adding New Images

1. Place image files in this directory
2. Use descriptive filenames
3. Update this README if adding new image types
4. Reference images in the main documentation

## Git Management

The `.gitignore` file is configured to:
- Keep documentation images in `docs/images/`
- Keep README images at the root level
- Ignore large media files in upload directories
- Maintain proper version control for documentation assets
